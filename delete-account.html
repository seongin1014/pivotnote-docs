PivotNote 계정 및 데이터 삭제

최종 수정: 2026-03-25

PivotNote 앱에서 계정 삭제 및 데이터 삭제를 요청하는 방법을 안내합니다.

방법 1: 앱 내에서 직접 삭제

회원 탈퇴 (계정 + 데이터 전체 삭제):

  1. PivotNote 앱을 엽니다
  2. 하단 탭에서 "더보기"를 탭합니다
  3. 화면 하단의 "회원 탈퇴"를 탭합니다
  4. 확인 팝업에서 "탈퇴"를 탭합니다
  5. 계정과 모든 데이터가 즉시 삭제됩니다

데이터 초기화 (계정 유지, 데이터만 삭제):

  1. PivotNote 앱을 엽니다
  2. 하단 탭에서 "더보기"를 탭합니다
  3. 화면 하단의 "데이터 초기화"를 탭합니다
  4. 확인 팝업에서 "초기화"를 탭합니다
  5. 모든 앱 데이터가 즉시 삭제됩니다

방법 2: 이메일로 삭제 요청

앱에서 직접 삭제할 수 없는 경우, 아래 이메일로 삭제를 요청할 수 있습니다.

이메일: contact@monik.kr

제목에 "계정 삭제 요청"을 포함하고, 가입 시 사용한 이메일 주소를 본문에 기재해 주세요. 요청 접수 후 30일 이내에 처리됩니다.

추가 보관 데이터

계정 삭제 후 추가로 보관되는 데이터는 없습니다. 모든 데이터는 삭제 요청 즉시 또는 이메일 요청 시 30일 이내에 완전히 파기됩니다.
